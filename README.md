# stock-api - WebAPI для отслеживания акций и комментирования
api представляет собой приложение, позволяющее добавлять в свой портфель акции разных компаний, а также некоторые социальные взаимодействия посредством комментирования разных акций.

# Endpoints
- api/account - управление аккаунтами: вход (авторизация + аутентификация) и регистрация.
- api/comment - управление комментариями, CRUD (базовые операции).
- api/portfolio - работа с портфолио пользователя.
- api/stock - работа с акциями.

# Используемые технологии
- .NET Core 8.0
- ASP.NET Core
- PostgreSQL 16
- Swagger/OpenAPI
- EFCore
- Microsoft.Aspnetcore.Identity
